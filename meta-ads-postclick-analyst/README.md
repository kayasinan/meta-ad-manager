# Post-Click Analyst (Agent 3)

**Location:** Machine B

Scores landing pages using GA4 engagement data to determine which URLs should be used in new campaigns.

## Responsibilities
- Pull GA4 landing page data filtered to Meta ad traffic
- Score pages by sessions, bounce rate, duration, CVR, revenue
- Classify as KEEP / FIX / KILL
- Produce approved URL list for Agent 6

## Outputs → Supabase
- Landing page scores and classifications
- Approved URL list
- Fix recommendations for underperforming pages
