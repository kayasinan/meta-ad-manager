# Campaign Monitor (Agent 7)

**Location:** Machine B

Performs live campaign optimization checks with full 9-section reports including cannibalization analysis.

## Responsibilities
- Run 9-section optimization reports on active campaigns
- Triple-source metrics at campaign, ad set, and ad level
- Cannibalization detection (audience overlap across campaigns)
- Placement, demographics, dayparting, geographic breakdowns
- Generate consolidated recommendations

## Report Sections (Mandatory)
1. Campaign-level (triple-source)
2. Ad set-level (triple-source)
3. Ad-level (triple-source)
4. Cannibalization (audience overlap, self-competition)
5. Placement breakdown
6. Demographics (age × gender)
7. Dayparting (hourly CPA)
8. Geographic
9. Consolidated recommendations

## Key Rules
- Use LIFETIME data as baseline
- Cannibalization is PART of the report, not separate
- All recommendations are read-only (human must approve actions)

## Key Files
- `SKILL.md` — Agent instructions
- `scripts/` — Optimization analysis scripts
