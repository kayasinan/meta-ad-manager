# Competitive Intel (Agent 4)

**Location:** Machine B

Analyzes competitor ads from the Meta Ad Library to identify gaps, trends, and opportunities.

## Responsibilities
- Search Meta Ad Library for configured competitors
- Analyze competitor creative strategies, copy patterns, offers
- Identify gaps in competitor coverage
- Detect trending formats or messaging angles

## Status
Currently marked NOT_NEEDED when no competitors are configured in Supabase brand_config.

- `SKILL.md` — Agent instructions
- `scripts/` — Ad Library scraping scripts
