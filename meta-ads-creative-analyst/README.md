# Creative Analyst (Agent 2)

**Location:** Machine B

Analyzes ad creative performance, scores fatigue levels, classifies winners, and produces replication blueprints for the Creative Producer.

## Responsibilities
- Analyze top 20 ads from Agent 1's output
- Score creative fatigue (HIGH/MODERATE/LOW) via weekly CTR trends
- Classify winners (ROAS ≥5x) and identify replication sources
- Bundle atomic creative units (image + URL + product + copy)
- Create never-produce list (themes/styles that consistently fail)

## Outputs → Supabase
- Fatigue scores per ad
- Winner classification
- Replication blueprint with source images
- Atomic units for Agent 5

- `SKILL.md` — Agent instructions
- `scripts/` — Analysis scripts
