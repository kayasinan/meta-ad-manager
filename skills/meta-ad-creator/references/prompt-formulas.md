# Prompt Formulas by Vertical

## Universal Structure

```
Professional [VERTICAL] advertisement for [BRAND],
[COLOR] solid color background,
[SUBJECT] prominently featured, photorealistic,
[PRODUCTS] displayed,
[BRAND] logo [POSITION],
headline text "[HEADLINE]" in bold modern sans-serif font,
[CTA] button in [CTA_COLOR],
[BADGE] promotional badge,
benefits listed with checkmark icons: [BENEFITS],
clean professional Meta ad format, square 1:1 aspect ratio,
high production value, studio quality
```

## Tips for Better Results

1. **"Solid color background"** prevents busy/distracting backgrounds
2. **"Photorealistic"** for product/lifestyle; skip for illustrated styles
3. **Always specify text exactly** — DALL-E 3 handles text well but needs explicit instructions
4. **Name specific breeds/models** — "Golden Retriever" > "dog"
5. **Include "Meta ad format"** — helps frame the composition correctly
6. **"Studio quality" or "high production value"** — pushes quality up
7. **Specify logo position** — "top-right" or "top-left" for consistency
